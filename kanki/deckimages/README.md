# Sample image files for KAnki

Place your card images in this folder. The images should be named according to the "image" field in your vocabulary cards.

For example:
- water.jpg (for the 水 card)
- book.jpg (for the 本 card)  
- house.jpg (for the 家 card)
- school.jpg (for the 学校 card)

Images can be JPG, PNG, GIF, or other web-supported formats.
Maximum recommended size: 500x500 pixels for best performance.

You can also use URLs for images by setting the "image" field to a full URL starting with "http" or "https".
